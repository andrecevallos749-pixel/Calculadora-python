# Calculadora Básica en Python parte 2

## Descripción

Esta segunda parte del proyecto consiste en la actualización y ampliación de la calculadora desarrollada en la etapa inicial. Se incorporaron nuevas funcionalidades y se mejoró la lógica del programa mediante la aplicación de operadores logicos, estructuras condicionales, bucles y validaciones de datos. Además, se actualizaron los diagramas de flujo para representar la evolución del sistema y su funcionamiento.

## Funcionalidades de la calculador actualizada

- Suma
- Resta
- Multiplicación
- División
- Validación de operaciones ingresadas por el usuario.
- Control de errores en divisiones por cero.
- Repetición de operaciones mediante bucles.
- Visualización clara de los resultados obtenidos.

## Herramientas utilizadas

- Python
- Visual Studio Code
- Git
- GitHub

## Actualizacion de todos los diagramas vistos en la parte 1

- Diagrama de flujo general de una calculadora basica actualizado y completo
- Diagrama IPO (Entrada - Proceso - Salida) actualizado y completo
- Diagrama de secuencia de interacción entre el usuario y el sistema actualizado y completo.

## Objetivo

Actualizar y ampliar la solución desarrollada en la primera etapa del proyecto, aplicando los conceptos fundamentales de lógica de programación para construir una calculadora más completa y estructurada. Para ello, se implementan operadores aritméticos, operadores lógicos, estructuras condicionales, bucles y mecanismos de validación, fortaleciendo la capacidad de análisis, diseño e implementación de algoritmos orientados a la resolución de problemas. Asimismo, se busca consolidar los conocimientos adquiridos durante el curso mediante la mejora progresiva del software y la incorporación de nuevas funcionalidades.
