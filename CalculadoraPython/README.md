# Calculadora Básica en Python

## Descripción

Este proyecto consiste en el desarrollo de una calculadora básica utilizando Python.

## Funcionalidades

- Suma
- Resta
- Multiplicación
- División

## Herramientas utilizadas

- Python
- Visual Studio Code
- GitHub

## Objetivo

Aplicar los conceptos de lógica de programación mediante estructuras condicionales, variables y validaciones de datos.
